# hospitality-analytics-dashboard
## 📌 Project Overview
This project analyzes hotel booking data to identify revenue trends, occupancy performance, and cancellation patterns. The dashboard helps management make data-driven decisions to improve profitability.

## 🎯 Business Objectives
- Monitor Total Revenue
- Analyze Occupancy Rate
- Track Cancellation Rate
- Understand Weekday vs Weekend performance
- Identify Revenue by State and Hotel
- Analyze Class-wise Revenue contribution

## 📊 Key Metrics Included
- Total Revenue  
- Occupancy %  
- Cancellation Rate  
- Total Bookings  
- Utilized Capacity  
- Revenue by State & Hotel  
- Class-wise Revenue  
- Checked-out, Cancelled & No-show analysis  
- Weekly Revenue & Booking Trends  

## 🛠 Tools Used
- Tableau (Data Visualization & Dashboarding)
- Excel (Data Cleaning & Preparation)
